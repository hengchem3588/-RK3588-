# -RK3588-
    校园安全巡检依赖人工，存在覆盖有限、夜间盲区多、响应滞后等问题。本作品基于 ELF 2 开发板，融合激光雷达、深度相机、IMU 等传感器，以 ROS 2 Humble 为控制框架，实现自主巡逻、安全检测、预警抓拍、远程上报的完整安防闭环。核心推理在端侧 NPU 完成，结合 QQ 远程交互与 DeepSeek 云端分析，形成"边缘推理 + 云端增强"协同架构。系统基于 slam_toolbox 实时建图，结合 Navigation 2 与 AMCL 实现航点循环巡逻与动态避障。板端部署 YOLOv8 RKNN 端侧推理，覆盖聚众斗殴、明火、香烟、路面坑洞等 12 类安全场景，NPU 本地完成不依赖云端。状态机驱动"巡逻→检测→报警→抓拍→通知→恢复"全流程闭环，自动记录日志并通过 pushplus 推送至移动端。可视化方面，基于 Python SimpleHTTP 与纯原生前端构建数据看板，包含饼图、柱状图及分类统计卡片等，后端调用 DeepSeek API 可一键生成安全分析报告，支持 MIPI 屏与手机浏览器局域网双端访问。远程交互方面，板端部署 OpenClaw 桥接 QQ，实现远程指令下发与现场图像回传。
